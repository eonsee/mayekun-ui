# 产品需求文档 - Mayekun 作品博客网站

## 1. 产品概述
Mayekun.com 是一个个人作品展示与博客平台，旨在为创作者提供一个优雅、专业的在线展示空间。网站支持文章发布与展示，帮助用户建立个人品牌形象。

- **目标用户**：个人创作者、设计师、开发者、内容创作者
- **核心价值**：简洁优雅的展示方式，专业的个人品牌塑造，流畅的阅读体验

## 2. 核心功能

### 2.1 功能模块
1. **首页**：Hero 展示区、导航栏、精选文章列表、作品展示
2. **文章列表页**：文章分类筛选、文章卡片展示、分页功能
3. **文章详情页**：文章内容展示、目录导航、相关推荐
4. **关于页面**：个人介绍、技能展示、联系方式

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero 展示区 | 个人品牌标语、动态背景效果、CTA 按钮 |
| 首页 | 导航栏 | 响应式导航菜单、Logo、主题切换 |
| 首页 | 精选文章 | 最新文章卡片展示、悬停动画效果 |
| 首页 | 作品展示 | 作品网格布局、分类标签、预览效果 |
| 文章列表页 | 分类筛选 | 文章分类标签、筛选动画 |
| 文章列表页 | 文章卡片 | 封面图、标题、摘要、发布时间、阅读量 |
| 文章详情页 | 文章内容 | Markdown 渲染、代码高亮、图片灯箱 |
| 文章详情页 | 目录导航 | 文章目录、滚动高亮、平滑滚动 |
| 文章详情页 | 相关推荐 | 相关文章推荐、标签关联 |
| 关于页面 | 个人介绍 | 头像、简介、社交链接 |
| 关于页面 | 技能展示 | 技能标签、进度条动画 |
| 关于页面 | 联系方式 | 联系表单、社交媒体链接 |

## 3. 核心流程

用户访问网站后，可以浏览首页了解博主信息，通过导航进入文章列表浏览文章，点击文章卡片进入详情页阅读完整内容。用户也可以通过关于页面了解博主详细信息并建立联系。

```mermaid
flowchart TD
    "访问首页" --> "浏览精选内容"
    "浏览精选内容" --> "进入文章列表"
    "浏览精选内容" --> "进入关于页面"
    "进入文章列表" --> "筛选分类"
    "筛选分类" --> "选择文章"
    "选择文章" --> "阅读文章详情"
    "阅读文章详情" --> "浏览相关推荐"
    "进入关于页面" --> "查看个人信息"
    "进入关于页面" --> "发送联系信息"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝色 (#1a1a2e) 作为主背景，搭配渐变紫蓝色 (#667eea → #764ba2) 作为强调色
- **辅助色**：柔和的灰白色系，营造高级感
- **按钮风格**：圆角按钮，带有微妙的阴影和悬停动效
- **字体**：
  - 标题：Playfair Display（优雅衬线体）
  - 正文：Noto Sans SC（中文友好）
- **布局风格**：卡片式布局，响应式网格，大量留白
- **图标风格**：线性图标，简洁现代

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | Hero 展示区 | 渐变背景、动态粒子效果、大标题动画、CTA 按钮悬停效果 |
| 首页 | 导航栏 | 固定顶部、毛玻璃效果、Logo 动画、响应式汉堡菜单 |
| 首页 | 精选文章 | 卡片网格、悬停放大效果、渐变边框、阴影层次 |
| 文章列表页 | 文章卡片 | 左侧封面图、右侧信息、标签徽章、阅读时间图标 |
| 文章详情页 | 文章内容 | 宽松行距、代码块主题、引用样式、图片懒加载 |
| 文章详情页 | 目录导航 | 固定侧边栏、滚动指示器、当前章节高亮 |
| 关于页面 | 个人介绍 | 居中布局、圆形头像、技能进度条动画 |

### 4.3 响应式设计
- **桌面优先**：优先设计桌面端体验，确保大屏幕下的视觉冲击力
- **移动端适配**：平板和手机端自动调整布局，导航折叠为汉堡菜单
- **触摸优化**：按钮和链接具有足够的点击区域，支持滑动手势

### 4.4 动效设计
- **页面加载**：渐入动画，内容依次显现
- **滚动触发**：元素进入视口时的淡入上滑效果
- **悬停交互**：卡片悬停时的缩放和阴影变化
- **主题切换**：平滑的明暗模式过渡动画
